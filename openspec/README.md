# openspec — Spec-Driven Development artifacts

This directory stores SDD change artifacts for the `opencode-pilot` project.

## Structure

```
openspec/
  changes/
    <change-name>/
      proposal.md
      spec.md
      design.md
      tasks.md
      apply-progress.md
      verify-report.md
      archive-report.md
      state.yaml
```

## Artifact store mode

`hybrid` — artifacts are persisted here AND in Engram for cross-session recovery.

## Usage

Each change lives in its own subdirectory under `changes/`. The `state.yaml` file
tracks the current phase and completion status of each artifact.
